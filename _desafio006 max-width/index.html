max-width

Utilizando max-width em vez de width, nesta situação, irá melhorar a maneira como o browser manipula janelas pequenas. Isto é importante quando se cria um site também para dispositivos móveis. Re-dimensiona esta página para verificares! A propósito, max-width é suportado por todos os browsers principais incluindo o IE7+ portanto não deves ter receio de utilizar.
